

```
mkdir pkoa

cd pkoa

npm init -y

npm i koa koa-router

npm i --save-dev typescript ts-node nodemon @types/node @types/koa @types/koa-router

# 生成 tsconfig.json
tsc --init
```

创建文件夹: `dist` 编译产物, `src` 源码

将tsconfig.json下的outDir路径修改为./build，rootDir路径修改为./src

```
"target": "es2017",
"outDir": "./dist",
"rootDir": "./src",
"allowJs": false,
"noImplicitAny": true,
```
